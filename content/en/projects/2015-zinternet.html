The Zinternet project - external link is an interactive drawing experience that I imagined as part of the development of the Zinefest website - external link in 2015.

year: 2015 date: 2015 contents_reading_time: seconds: 32 text: >- 1 minute contents: - kind: block template: chapter title: >- slug: >- ranks: self: 2 data: layout: no_background text: >-

Pour mes études, je devais réaliser un site web, et je n’aime pas les choses qui ne servent à rien d’autre qu’à obtenir une note. J’ai donc proposé à Disparate - external link, qui organise ce festival de micro-édition, de réaliser gratuitement le site web du festival.

L’idée derrière Zinternet est une page blanche sur laquelle tous les visiteurs du site peuvent dessiner. Les dessins produits par les visiteurs du site sont également postés sur le Tumblr - external link du projet. Ces dessins sont ensuite publiés sous forme de fanzine disponible à prix libre en soutien au festival Zinefest.

Aujourd’hui, il continue d’accompagner le festival Zinefest année après année, en accueillant des dessins pour chaque nouvelle édition.
